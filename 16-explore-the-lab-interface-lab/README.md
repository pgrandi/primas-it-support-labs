# Steps Performed

1. Accessed the lab environment.
2. Navigated from the networking closet to Office 2.
3. Added a monitor to the workstation.
4. Connected the monitor to the computer using an HDMI cable.
5. Connected the monitor power cable to the wall outlet.
6. Added keyboard and mouse peripherals.
7. Connected keyboard and mouse to USB ports on the computer.
8. Powered on the monitor and workstation.

---

# Troubleshooting Notes

## HDMI Connection
- Verified HDMI cable was connected to the correct video output port.

## Power Connection
- Confirmed monitor power cable was connected to both the monitor and outlet.

## Peripheral Setup
- Verified keyboard and mouse were connected to available USB ports.
