# Lab 08: Upgrade a Video Card

## Objective

Upgrade a computer's graphics card to support high-definition video playback and modern display connectivity. Remove the existing GPU, install a new PCIe graphics card, connect supplemental power, and connect the monitor using HDMI.

## Skills Demonstrated

- GPU installation and removal
- PCIe expansion card configuration
- HDMI display connectivity
- Internal hardware upgrades
- Power supply connections
- Display troubleshooting
- Video interface identification
- Desktop computer hardware upgrades

## Lab Tasks

Completed the following tasks:

1. Removed the existing VGA video cable.
2. Removed the existing VGA PCI video adapter.
3. Installed a modern PCIe video card with HDMI output.
4. Connected supplemental PCIe power to the graphics card.
5. Connected the monitor using an HDMI cable.
6. Verified successful video output and system functionality.

## Technologies Used

- TestOut LabSim
- PCIe Graphics Card
- HDMI Display Connection
- VGA Video Adapter
- ATX Desktop Computer
- PCIe Power Connector
- Monitor and Display Interfaces
