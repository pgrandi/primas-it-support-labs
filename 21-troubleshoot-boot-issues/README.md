# CompTIA A+ Lab 4.2.6 – Troubleshoot Boot Issues

## Objective
Diagnose and resolve boot issues that occurred after a power supply replacement by identifying missing power and data connections and restoring normal system operation.

## Skills Demonstrated
- Boot Troubleshooting
- Hardware Diagnostics
- Power Supply Installation Verification
- SATA Power Configuration
- SATA Data Cable Configuration
- Motherboard Power Verification
- Desktop Hardware Support
- System Startup Verification

## Tasks Completed
1. Inspected the computer after a power supply replacement.
2. Connected the CPU power connector to the motherboard.
3. Identified a missing SATA power connection.
4. Connected the SATA power cable to the hard drive.
5. Identified a missing SATA data connection.
6. Connected the SATA data cable between the motherboard and hard drive.
7. Turned on the power supply switch.
8. Powered on the computer and verified a successful boot into Windows.
