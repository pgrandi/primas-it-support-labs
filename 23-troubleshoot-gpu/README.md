# CompTIA A+ Lab 4.3.9 – Troubleshoot GPU

## Objective
Diagnose and resolve a display issue caused by an improperly powered graphics card. Verify GPU connectivity and restore video output to the monitor.

## Skills Demonstrated
- GPU Troubleshooting
- Graphics Card Installation Verification
- PCIe Power Connector Configuration
- Hardware Diagnostics
- Display Troubleshooting
- Desktop Hardware Support
- Video Output Verification
- System Startup Troubleshooting

## Tasks Completed
1. Powered on the computer and verified that no display signal was present.
2. Inspected monitor and video cable connections.
3. Opened the system case and examined the graphics card installation.
4. Identified that the GPU auxiliary PCIe power connector was not connected.
5. Connected the 6-pin PCIe power cable from the power supply to the graphics card.
6. Powered on the system and verified proper video output.
7. Confirmed successful display functionality and normal system operation.
