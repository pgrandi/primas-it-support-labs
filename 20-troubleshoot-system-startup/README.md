# CompTIA A+ Lab 4.2.4 – Troubleshoot System Startup

## Objective
Diagnose and correct startup issues after a motherboard replacement. The goal was to identify hardware-related problems preventing the system from booting and restore normal operation.

## Skills Demonstrated
- Hardware troubleshooting
- Motherboard component verification
- CPU cooling system inspection
- Power connection validation
- Startup diagnostics
- System boot verification

## Tasks Completed
1. Inspected the system after motherboard replacement.
2. Verified power connections to the computer and monitor.
3. Identified that the CPU fan was not connected to the motherboard.
4. Connected the CPU fan to the proper CPU_FAN header.
5. Powered on the system and verified successful startup.
6. Confirmed the operating system loaded successfully.
