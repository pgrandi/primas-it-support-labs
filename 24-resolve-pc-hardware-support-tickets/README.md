# CompTIA A+ Lab 4.4 – Resolve PC Hardware Support Tickets

## Objective
Troubleshoot and resolve multiple hardware-related support tickets by diagnosing startup, display, and memory issues and restoring normal system functionality.

## Skills Demonstrated
- Hardware Troubleshooting
- IT Support Ticket Resolution
- PC Startup Diagnostics
- Memory Installation and Verification
- Monitor Troubleshooting
- Display Connectivity Troubleshooting
- End User Support
- Desktop Hardware Support
- Root Cause Analysis
- Technical Documentation

## Tasks Completed

### Ticket 11 – Executive Office
1. Reviewed the support ticket regarding a computer that would not power on.
2. Inspected the workstation and peripheral equipment.
3. Identified that the monitor power switch was turned off.
4. Powered on the monitor and verified proper system operation.

### Ticket 12 – Office 2
1. Reviewed the support ticket regarding a computer emitting beep codes during startup.
2. Opened the system case and inspected internal hardware.
3. Determined that system memory was not installed.
4. Installed the RAM modules correctly.
5. Powered on the system and verified successful Windows startup.

### Ticket 13 – Office 1
1. Reviewed the support ticket regarding a monitor displaying no image.
2. Inspected display cable connections.
3. Identified that the HDMI cable was not connected to the computer.
4. Connected the HDMI cable properly.
5. Verified successful display output and Windows operation.

### Ticket Documentation
1. Documented actions taken for each support ticket.
2. Updated ticket status.
3. Closed all completed tickets.
