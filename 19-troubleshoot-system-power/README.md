# Troubleshoot System Power

## Objective
Diagnose and resolve power-related issues preventing a workstation from booting by examining physical connections and power supply settings.

## Skills Demonstrated
- Power Supply Troubleshooting
- Hardware Diagnostics
- Power Connection Verification
- PSU Voltage Configuration
- Desktop Hardware Support
- IT Help Desk Ticket Resolution
- System Startup Verification
