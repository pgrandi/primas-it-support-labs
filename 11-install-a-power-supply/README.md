# CompTIA A+ Lab 3.1.2 - Install a Power Supply

## Objective
Replace and install a power supply while ensuring the system has the required motherboard, CPU, SATA, and PCIe power connections.

## Tasks Performed

- Selected an appropriate replacement power supply
- Installed the power supply in the ATX case
- Connected the 24-pin motherboard power connector
- Connected the 8-pin CPU power connector
- Connected SATA power connectors to three hard drives
- Connected the AC power cable
- Turned on the power supply
- Powered on the computer successfully

## Skills Demonstrated

- Power supply replacement
- Motherboard power connections
- CPU power connections
- SATA power connections
- PC hardware installation
- Desktop computer assembly
- Hardware troubleshooting

## Result

Lab completed successfully with a score of 100%.
